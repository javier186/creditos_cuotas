La integración con Google Sheets requiere Google Apps Script. Sigue la guía de abajo para configurarla gratis en menos de 10 minutos. Los datos se sincronizan automáticamente cada vez que presionas "Sincronizar".
Guía paso a paso: conectar Google Sheets
1
Crea una hoja de Google Sheets
Ve a sheets.google.com, crea una hoja nueva y cópiala la URL. El ID es la parte entre /d/ y /edit.
Ej: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5.../edit
2
Abre el editor de Apps Script
En tu hoja de Google Sheets, ve al menú Extensiones → Apps Script. Esto abre el editor de código.
3
Pega este código en el editor
Borra lo que hay, pega el siguiente código y guarda (Ctrl+S):
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); // Hoja de Clientes var sh = ss.getSheetByName('Clientes') || ss.insertSheet('Clientes'); if (sh.getLastRow() === 0) { sh.appendRow(['ID','Nombre','Teléfono','Email', 'Máquina','Total','Inicial','Fecha venta', 'Cuotas pagadas','Cuotas totales','Estado']); } sh.clearContents(); sh.appendRow(['ID','Nombre','Teléfono','Email', 'Máquina','Total','Inicial','Fecha venta', 'Cuotas pagadas','Cuotas totales','Estado']); data.clientes.forEach(function(c) { var pagadas = c.cuotas.filter(function(q){return q.estado==='pagado';}).length; var estado = pagadas === c.cuotas.length ? 'Finalizado' : 'Activo'; sh.appendRow([c.id, c.nombre, c.tel, c.email, c.maquina, c.total, c.inicial, c.fecha, pagadas, c.cuotas.length, estado]); }); // Hoja de Cuotas var sh2 = ss.getSheetByName('Cuotas') || ss.insertSheet('Cuotas'); sh2.clearContents(); sh2.appendRow(['Cliente','Cuota #','Fecha','Monto','Estado']); data.clientes.forEach(function(c) { c.cuotas.forEach(function(q) { sh2.appendRow([c.nombre, q.num, q.fecha, q.monto, q.estado]); }); }); return ContentService .createTextOutput(JSON.stringify({ok: true})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok:false,error:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService .createTextOutput('CaféFinance Webhook activo ✓') .setMimeType(ContentService.MimeType.TEXT); }
4
Despliega como Web App
Haz clic en Implementar → Nueva implementación.
• Tipo: Aplicación web
• Ejecutar como: Yo
• Quién tiene acceso: Cualquier usuario
Clic en Implementar y copia la URL que aparece.
5
Pega la URL en CaféFinance
Copia la URL del paso anterior y pégala abajo. Luego prueba la sincronización.
